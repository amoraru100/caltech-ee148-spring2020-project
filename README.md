# Caltech EE148 Final Project Spring 2020 
## Alex Moraru and Jessica Wang
<br />

Our computer vision project for detecting and classifying a vehicle + estimating a damaged vehicle's fair price

### .annotations/
directory containing pre-processed and processed data (.csv files)

### .models/
directory containing all saved classification models (i.e. modified ResNet18 and ResNet34)

### .predictions/
directory containing predicted bounding boxes from detector (.json files)

### .splits/
directory containing indices of different training and validation splits (.npy files)

### .visualizations/
directory containing plots and visualizations (i.e. confusion matrix, annotated examples of data)
